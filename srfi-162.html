Title

Comparators sublibrary

Author

John Cowan

Status

Abstract

This SRFI provides a few extra procedures and comparators to go with SRFI 128, Comparators. Implementers are urged to add them to their SRFI 128 libraries, for which reason they are not packaged as a separate library.

Issues

None at present.

Rationale

The comparator-distinct? procedure is taken from the Haskell Eq typeclass. The comparator-max and comparator-min procedures are taken from the Haskell Ord typeclass. The useful comparators are described, but not named, in SRFI 128 itself.

Specification

Definitions

Predicates

(comparator-distinct? comparator obj1 obj2)

Returns #t if obj1 and obj2 are not the same in the sense of the equality predicate of comparator.

(comparator-max? comparator obj1 obj2 ...)

Returns the largest obj in the sense of the ordering procedure of comparator.

(comparator-min? comparator obj1 obj2 ...)

Returns the smallest obj, in the sense of the ordering procedure of comparator.

Useful comparators

These comparators are useful for comparing standard Scheme types, and should be more efficient than default comparators.

boolean-comparator

A comparator for booleans, such that #f compares before #t.

real-comparator

A comparator for real numbers.

string-comparator

A comparator for strings using the implementation-specific definition of string.

string-ci-comparator

A comparator for strings using the implementation-specific definition of string-ci.

list-comparator

A comparator for lists that uses default-comparator to compare the elements.

vector-comparator

A comparator for vectors that uses default-comparator to compare the elements.

Implementation

The sample implementation is (not yet) found in the repository of this SRFI. It contains the following files:

  • predicates-impl.scm - an implementation of the predicates
  • useful-comparators.scm - an implementation of the comparators
  • extra-tests.scm - tests for the SRFI 162 additions

Make sure when merging these into SRFI 128 that they are properly exported and that useful-comparators.scm is included after the definitions of the SRFI 128 procedures.
